# פרופוליו - תיק עבודות מקצועי בעברית

## 🎯 תיאור הפרויקט

פרופוליו היא פלטפורמה מקצועית לבניית תיקי עבודות מדהימים לאנשי מקצוע, עסקים מקומיים ומספקי שירותים. האתר בנוי בעברית עם תמיכה מלאה ל-RTL (Right-to-Left) וכולל אנימציות חלקות, ביקורות לקוחות, ותמונות לדוגמה מקצועיות.

## ✨ תכונות עיקריות

### עיצוב מקצועי
- **עיצוב מינימליסטי מודרני** - ממשק נקי וברור שמעביר אמון וקרדיביליטי
- **צבעים מקצועיים** - כחול כהה (#1a3a52) כצבע ראשי וזהב חם (#d4a574) כהדגשה
- **טיפוגרפיה Rubik** - גופן מודרני וקריא לכל התוכן

### תכונות טכניות
- **RTL מלא** - תמיכה מלאה בעברית עם כיוון מימין לשמאל
- **אנימציות חלקות** - מעברים וחיות אינטראקטיביות בכל הרכיבים
- **רספונסיבי** - עובד בצורה מושלמת בכל גדלי המסכנים
- **ביקורות לקוחות** - קרוסלה של ביקורות אמיתיות עם דירוגי כוכבים
- **תמונות לדוגמה** - 4 תמונות מקצועיות המציגות תיקי עבודות שונים

## 🚀 התחלה מהירה

### דרישות מוקדמות
- Node.js 18+
- pnpm (מומלץ) או npm

### התקנה

```bash
# התקן את התלויות
pnpm install

# הפעל את שרת הפיתוח
pnpm dev

# בנה את הפרויקט לייצור
pnpm build
```

שרת הפיתוח יהיה זמין בכתובת `http://localhost:3000`

## 🎨 עיצוב ותכנון

### פילוסופיית העיצוב
הפרויקט עוקב אחר עיצוב **מינימליסטי מקצועי** עם דגש על:
- **בהירות** - ממשק נקי וקל להבנה
- **אמון** - צבעים מקצועיים וטיפוגרפיה קריאה
- **אנימציות** - מעברים חלקים שמחזקים את חוויית המשתמש
- **RTL-aware** - כל הרכיבים מותאמים לעברית

### צבעים
- **Primary**: #1a3a52 (כחול כהה) - אמון ומקצועיות
- **Accent**: #d4a574 (זהב חם) - איכות פרימיום
- **Background**: #ffffff (לבן) - בהירות
- **Text**: #2c3e50 (צבא כהה) - קריאות

### גופנים
- **Display**: Rubik Bold (700) - כותרות
- **Body**: Rubik Regular (400) - טקסט גוף
- **Accent**: Rubik Medium (500) - הדגשות

## 🎬 אנימציות

### Hover Effects
- **כפתורים**: קנה מידה קטן (0.98) עם צל מורחב
- **כרטיסים**: הרמה עם צל מוגבר
- **תמונות**: הגדלה (1.1x) עם מעבר חלק

### Entrance Animations
- אלמנטים נכנסים עם fade-in חלק
- סטיגר של 50-100ms בין אלמנטים

### Transitions
- משך ברירת מחדל: 200-300ms
- Easing: cubic-bezier(0.23, 1, 0.32, 1) (ease-out)

## 🌐 RTL Support

כל הרכיבים מותאמים לעברית:
- `dir="rtl"` בתג `<html>`
- `lang="he"` להגדרת השפה
- כל הטקסט מיושר לימין
- אייקונים מהופכים בעת הצורך

## 🤝 תמיכה וטיפול

### בעיות נפוצות

**הטקסט לא מיושר לימין:**
- וודא שה-`dir="rtl"` קיים בתג `<html>`

**אנימציות לא עובדות:**
- בדוק את הדפדפן - עדכן לגרסה אחרונה
- וודא שאין הגדרת `prefers-reduced-motion`

**תמונות לא מופיעות:**
- בדוק את ה-URLs של התמונות בקובץ `Home.tsx`
- וודא שיש חיבור לאינטרנט
