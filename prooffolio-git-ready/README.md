# Prooffolio Hebrew Landing Page

אתר סטטי בעברית, מוכן ל-GitHub ול-Vercel.

## הרצה מקומית

```bash
npm install
npm run dev
```

פתחו בדפדפן: http://localhost:3000

## בנייה לפרודקשן

```bash
npm run build
```

הקבצים המוכנים ייווצרו בתיקיית `dist`.

## העלאה ל-GitHub

```bash
git init
git add .
git commit -m "Initial Hebrew landing page"
git branch -M main
git remote add origin YOUR_REPOSITORY_URL
git push -u origin main
```

## Vercel

Vercel יריץ אוטומטית:

```bash
npm run build
```

וישרת את תיקיית `dist`.
